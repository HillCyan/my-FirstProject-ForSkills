# 对话总结

这个 Skill 帮助用户总结对话内容，提取关键信息。

## 使用场景

- 用户请求"总结一下我们的对话"
- 用户想回顾讨论的内容
- 用户需要记录会议或讨论要点
- 对话较长，需要提炼核心内容

## 功能

1. **关键内容提取**：总结对话中讨论的主要话题
2. **决策记录**：记录对话中做出的决定
3. **待办事项**：列出需要后续跟进的任务
4. **问题记录**：记录尚未解决的问题

## 输出格式

```markdown
## 对话总结

### 主要话题
- 话题1
- 话题2

### 关键决策
- 决策1
- 决策2

### 待办事项
- [ ] 任务1
- [ ] 任务2

### 待解决问题
- 问题1
- 问题2
```

## 注意事项

- 总结应简洁明了
- 突出重点，省略无关细节
- 如有需要，可以按时间线整理
